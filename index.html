O Pensamento Computacional é uma habilidade essencial, pois permite resolver problemas de maneira estruturada, entender algoritmos e aplicar soluções lógicas para tarefas cotidianas. Para mim, é importante aprender a desmembrar problemas em etapas menores e usar o raciocínio lógico para chegar a soluções eficientes.

  • Compreendi como usar tags básicas de HTML, como <h1>, <p>, <a>.
  • Aprendi a estilizar páginas com CSS, como alterar cores de texto, fundos e fontes.
  • Entendi o conceito de layout responsivo e como utilizar flexbox para alinhamento de elementos.
